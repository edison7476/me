Forge

This is a game seraching and streaming web application that allows users to search games, preview the selected game playing by the most popular streamer on Twitch, and also display the matching Amazon product listings of a selected game on the site. Working in a team of 3 peers, we accomplished the application from scratch in 3 days using Node.js, Express.js, Mongoose.js, Angular, Twitch API and Amazon Product Advertisement API.

Mini Store

In this mini store web application, I created a REST API using Angular.js, Express.js, Node.js, and MongoDB that demonstrates different types of accessibility based on the level of authority.
Admin level users are able to pull/create/update/delete information from the database such as listing products, reviewing all the customers information and order history, and deleting listed products while normal users are only able to access their own information, place orders and write product reviews.
